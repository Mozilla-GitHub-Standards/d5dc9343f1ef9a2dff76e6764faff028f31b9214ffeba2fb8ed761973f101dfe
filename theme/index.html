{{site.title}} Tool

Many open source projects now use a Code of Conduct ("CoC") to define their standards for community behavior. While many are crafted with care, and intent - far more (for a number of reasons)are invalid, and thus pose harm. We created this tool to promote the development of strong Code of Conducts in Open Source, and to help potential contributors evaluate projects for participaiton.

We warmly welcome contributions to this resource.
